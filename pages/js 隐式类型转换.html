    • NaN == NaN: 返回 false， 特殊例子
    • 引用类型 == 引用类型
      • 两个引用类型对比时，比较内存地址， 如果相同就相同
  • 两边类型不同时
    • （null || undefined || NaN）== (Object || String || Number || Boolean ): 不做隐式转换
      • 例子
        // 无转换过程， 直接比较

    • Boolean == ? : 把 Boolean 转成数值（0 或 1）
      • 例子
        true == 32 // 转换过程： 1 == 32
true == 1   // 转换过程： 1 == 1
false == 0 // 转换过程： 0 == 0
    • String == Number : 字符转成数值
      • 例子
        1 == '123' // 转换过程： 1 == 123
123 == '123' // 转换过程： 123 == 123
0 == '0' // 0 == 0
1 == '1' //  1 == 1
    • Object == (Number||String): 对象返回默认值，通过 先valueOf，再 toString 将对象转成原始值(Number||String)， 如果转换不了就报错

      简写个伪代码： !Number(引用类型.valueOf())?.toString()

      1. 引用类型调用 valueOf
      1. 把 （1）的值转成数值
      1. 如果（2）结果是数值，就用数值比较；
      1. 如果（2）结果不是数值， 把（1）结果使用 toString 转换;
      1. 如果（4）toString 转换不了，直接报错
      • 例子
        //4. 对象转换不是数值

b === 123 // b.valueOf() === 123

// (B). '123' 转换过程： 再流程 （A） 得到 123:      123 == '123'
// (C). 回到 String == Number 的类型对比，隐士转换得到 =>  123 == 123 // true

隐士转换过程

toPrimity 先 valueOf() 再 toString()
Date 是 先调用 toString() 再 valueOf()
  1. {...} + ??: 不管前面的大括号，认为其是块级语法；
    • 例子
      {} + [] // 0 
{a: 1} + [] // 0
{a: 1} + [2] // 2

new Date() + [3] // 'Sun Dec 19 2021 18:34:34 GMT+0800 (中国标准时间)3'
  1. Number + Number： 按照数值计算， 优先级高
  1. String + Number : Number 转成字符进行相加，优先级中
  1. 不是 Number + 不是 String ，优先级低
    1. 如果能转成数值， 使用数值相加
      • 例子
        true + false // 1
true + true // 2
false + false // 0
2 + null // 2
2 + undefined // NaN
2 + true // 3

undefined + 2 // 'undefined2'
null + 2 // 'null2'
    1. 如果（1） 不成立， 调用 ??.valueOf() 得到基础类型（Number||Sring||Boolean||null||undefined ）,进行处理
    1. 如何（2）不成立，调用 ??.toString() 得到基础类型（Number||Sring||Boolean||null||undefined ），进行处理
    1. 如果（3）不成立，抛出报错

xxx - yyy 隐士转换

  • 数值类型： 直接相减
  • 引用类型
    1. {...} - ??: 与加法一样，不管对象里的值，其被认为是块级元素
      • 例子
        {} - [2] // -2
{} - [] // -0
    1. 引用类型: 转成数值， 通过数值计算

toPrimitive

  • 非基础类型通过 valueOf 转成基础类型, 如果不能就使用 toString 转换
  • js基础类型: string number boolean null object function, 新增 symbol
  • Symbol.toPrimitive 把对象转换成为基础类型

Symbol.toPrimitive

hint 值是3种: "number"、"string" 和 "default"

var object2 = {}
console.log(`${object2}`); // "[object Object]"

// object 转换被复写操作
const object1 = {
  [Symbol.toPrimitive](hint) {
	console.log('hint is :', hint)
    if (hint === 'number') {
      return 42;
    }
  }
};

console.log(+object1); // expected output: 42
console.log(`${object1}`); // string
console.log('' + object1);// default 
console.log(object1+''); // default
console.log(!object1);
console.log(!!object1);
